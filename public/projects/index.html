我的项目

以下是我参与或独立开发的一些项目。

项目一：个人博客系统

  • 描述：基于 Hexo 搭建的静态博客系统，支持 Markdown 写作、主题定制和 GitHub Pages 部署。
  • 技术栈：Hexo、Node.js、EJS、Stylus
  • 链接：GitHub 仓库

项目二：在线待办清单

  • 描述：一个简洁的在线待办事项管理工具，支持添加、删除、标记完成等功能。
  • 技术栈：Vue.js、Vuex、LocalStorage
  • 链接：在线体验

项目三：天气预报小程序

  • 描述：基于微信小程序开发的天气预报应用，支持定位获取实时天气和未来预报。
  • 技术栈：微信小程序、JavaScript、REST API
  • 链接：微信小程序搜索“天气预报”
